Shooting Game

Introduction

This is a console game written in c++.
The gameplay of it is as the followings :
The player, marked by X, can shoot bullets, to destroy enemys.
However, the player should avoid use up all bullets, or it will die easily.
Your target is to defeat the enemies to earn score, and survive till the 100th wave. If you think the game was too easy/difficult, you can choose the mode that you want, to make it sutable for you.
